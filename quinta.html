¿Qué es una tabla en HTML?

Una tabla en HTML se crea usando las etiquetas básicas:

  • <table>: contenedor de la tabla.
  • <tr>: representa una fila.
  • <th>: representa una celda de encabezado.
  • <td>: representa una celda de datos.

¿Cómo darle estilo a una tabla con CSS?

Con CSS puedes cambiar colores, bordes, fuentes, alineación, espaciado, etc. Por ejemplo:

table {
}

¿Sabías que?

HTML te permite organizar visualmente datos complejos, CSS lo hace más atractivo y JavaScript le da interactividad. ¡Combínalos para crear experiencias increíbles!
